# Apirestclase
Ejemplo de api en Slim (Basico)

Pasos para Instalar
1- Descargar o mediante git clonar en la carpeta donde se instalara el proyecto
2- Desplazarse con la consola (CMD) hasta la carpeta.
3- Ejecutar composer install
